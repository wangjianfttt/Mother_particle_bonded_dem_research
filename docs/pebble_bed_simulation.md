# 球床压缩破碎模拟方案

## 1. 基本思路

将已经标定的 Li4SiO4 bonded cluster 作为可破碎母颗粒模板，随机注入到盒体或圆柱容器中，经过沉降和压实后，使用上压板进行准静态一维压缩。随着压缩位移增加，记录 bond 断裂、颗粒破碎、碎片重排和球床宏观力学性质演化。

## 2. 算例分级

### Case B0：不可破碎球床

所有母颗粒作为普通不可破碎 DEM 球，用于获得基准应力-应变曲线、配位数和孔隙率演化。

### Case B1：小规模可破碎球床

- 母颗粒数量：20-100；
- 每个母颗粒：约 500 子颗粒；
- 目标：观察破碎机制和后处理流程是否可靠。

### Case B2：中等规模可破碎球床

- 母颗粒数量：100-500；
- 每个母颗粒：约 500 子颗粒；
- 目标：得到具有统计意义的破碎概率、空间分布和宏观响应。

### Case B3：对照简化破碎模型

使用预设破碎颗粒比例或 crush-probability 方法，与真实 BPM 可破碎模型对比。

## 3. 关键控制变量

- 初始填充率；
- 母颗粒粒径分布；
- 母颗粒强度分布；
- 颗粒-颗粒摩擦系数；
- 墙面摩擦系数；
- 上压板压缩速率；
- 压缩最大应变；
- 容器边界条件；
- 是否循环加载。

## 4. 输出指标

### 宏观指标

- 应力-应变曲线；
- 等效压缩模量；
- 床层高度变化；
- 总孔隙率；
- 残余变形；
- 能量耗散。

### 细观指标

- 配位数；
- 接触力分布；
- 强力链空间结构；
- bond 断裂数量；
- bond 断裂位置；
- 母颗粒破碎比例；
- 破碎母颗粒空间分布；
- 碎片粒径分布；
- 碎片是否填隙或迁移。
