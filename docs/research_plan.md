# 基于 Bonded-Particle DEM 的 Li4SiO4 球床颗粒破碎研究计划

## 1. 研究目标

聚变堆固态增殖包层中的 Li4SiO4 陶瓷球床会在装填、热膨胀、约束压缩和循环运行过程中承受复杂接触载荷。颗粒破碎会改变球床的孔隙率、配位数、力链、等效刚度、渗透性和导热路径，因此需要建立能够从单颗粒破碎传递到球床尺度的 DEM 模型。

本课题拟采用 LIGGGHTS-INL 的 bonded-particle model，将单个 Li4SiO4 陶瓷球离散为 500 个以上小颗粒并用 bond 连接。研究分为单颗粒标定和球床压缩破碎两个层级。

## 2. 核心科学问题

1. 如何用 bonded-particle model 复现 Li4SiO4 单颗粒的弹性响应、crush load 和强度离散性？
2. 单颗粒 BPM 微观参数，包括 bond 刚度、bond 强度、bond 半径和非均质分布，如何影响宏观压碎行为？
3. 可破碎颗粒注入球床后，局部接触力链如何诱发颗粒破碎？
4. 颗粒破碎如何改变球床压缩应力-应变曲线、孔隙率、配位数、力链分布和碎片粒径分布？
5. 相比不可破碎 DEM 或预设破碎比例 DEM，真实 bond 断裂模型能揭示哪些额外机制？

## 3. 总体技术路线

### 阶段 A：文献数据整理

- 汇总 Li4SiO4 单颗粒压碎实验，包括球径、压板材料、加载速率、平均 crush load、离散性和 Weibull 参数。
- 汇总 ceramic breeder pebble bed 一维压缩实验和 DEM 模拟，包括填充率、床层尺寸、摩擦系数、压缩应变范围和宏观模量。
- 汇总 bonded-particle model 标定方法，尤其是陶瓷、岩石、矿石和脆性颗粒破碎。

### 阶段 B：单颗粒 BPM 建模与标定

1. 生成近似球形的 bonded cluster。
2. 设置子颗粒材料参数、接触模型和 bond 模型。
3. 用上下刚性板进行单颗粒压缩。
4. 输出力-位移曲线、首个 bond 断裂载荷、峰值载荷、破碎模式和碎片分布。
5. 多目标标定：
   - 弹性段斜率匹配实验等效刚度；
   - 峰值载荷匹配实验 crush load；
   - bond 强度非均质性匹配 crush load 离散性；
   - 分辨率敏感性验证 500 子颗粒模型是否足够稳定。

### 阶段 C：球床可破碎模板注入

- 将标定好的 bonded cluster 作为单颗粒模板。
- 生成包含多个 Li4SiO4 可破碎母颗粒的球床。
- 先做小规模机制算例，再做中等规模统计算例。
- 初始阶段采用室温、准静态、一维压缩，不加入热耦合。

### 阶段 D：球床压缩破碎分析

主要输出：

- 宏观应力-应变曲线；
- 床层等效压缩模量；
- 孔隙率演化；
- 配位数演化；
- 接触力分布和强力链；
- bond 断裂数量与断裂速率；
- 破碎母颗粒数量和空间分布；
- 碎片粒径分布；
- 与不可破碎 DEM 和预设破碎 DEM 的对照。

## 4. 建议研究边界

第一篇文章建议只做力学压缩与颗粒破碎，不直接加入热-流-固全耦合。推荐题目：

> 基于 bonded-particle DEM 的 Li4SiO4 陶瓷球单颗粒压碎标定及球床压缩破碎演化

这样范围足够聚焦，验证链条也更清楚。

## 5. 风险与控制

### 计算规模风险

一个母颗粒包含 500 个子颗粒时，1000 个母颗粒对应约 50 万个 DEM 颗粒，并包含大量 bond，计算成本较高。

控制方案：

- 单颗粒标定使用 500、800、1500 子颗粒做分辨率对比。
- 球床机制研究先用 20-100 个母颗粒。
- 中等规模统计研究使用 100-500 个母颗粒。
- 大规模球床可先采用局部可破碎策略。

### 参数非唯一性风险

不同 bond 刚度、bond 半径和强度组合可能得到相似 crush load。

控制方案：

- 不只匹配峰值载荷，还匹配弹性刚度、破碎模式和碎片粒径分布。
- 使用多尺寸单颗粒实验作为约束。
- 引入 Weibull 或空间随机场描述材料非均质性。

### 准静态加载风险

加载速率过高会引入惯性效应。

控制方案：

- 监控动能/总能量或动能/弹性能比例。
- 做加载速率敏感性分析。
- 采用足够阻尼，但避免过度阻尼改变破碎行为。

## 6. 初步时间表

- 第 1-2 个月：文献数据表、单颗粒模板生成、LIGGGHTS-INL 单颗粒压缩跑通。
- 第 3-4 个月：单颗粒 BPM 参数标定和敏感性分析。
- 第 5-6 个月：小规模可破碎球床压缩算例。
- 第 7-8 个月：球床破碎统计、力链和孔隙率演化分析。
- 第 9-10 个月：对照模型和参数研究。
- 第 11-12 个月：论文图表和初稿。
