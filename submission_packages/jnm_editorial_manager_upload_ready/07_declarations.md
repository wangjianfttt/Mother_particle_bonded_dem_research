## Data availability statement for Editorial Manager

The processed event tables, native force-network summaries, macro-topology-event metrics, figure source data, manuscript figures and post-processing scripts supporting this study are assembled in a reduced reproducibility package with a checksum manifest. The package will be deposited in a persistent repository before final submission, and the generated repository DOI or stable URL should be inserted in the manuscript and submission system. Very large raw restart files and complete raw local-bond dump histories are retained as local case archives and can be provided as larger audit bundles on reasonable request, subject to repository file-size limits and institutional storage constraints.

## Code availability statement for Editorial Manager

The template-generation, run-control, post-processing, figure-building and manuscript-building scripts supporting the submitted results are included in the reduced reproducibility package under `scripts/`. Representative DEM input files and processed outputs are included where needed to audit the manuscript-level claims.
