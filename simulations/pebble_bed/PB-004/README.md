# PB-004 calibrated weak-plane pebble-bed compression

PB-004 embeds the SP-002-CAL1 weak-plane bonded 1 mm Li4SiO4 pebble template into a compact 2 x 2 x 3 bed.

Each mother pebble contains 500 subparticles. Particle types 1 and 2 are the two sides of the internal weak plane; wall meshes are type 3. Cross-plane particle-particle bonds use reduced strength and reduced bond creation distance, while intra-side bonds use the calibrated bulk strength.

Initial goal: establish the end-to-end workflow for calibrated-template bed compression and per-pebble breakage-event sequencing before scaling to larger beds.
