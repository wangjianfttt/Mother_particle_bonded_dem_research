# PB-005 36-pebble calibrated-template bed

PB-005 scales the PB-004 calibrated-template workflow from 12 mother pebbles to a 3 x 3 x 4 ordered bed with 36 mother pebbles.

- Mother pebbles: 36
- Subparticles per mother pebble: 500
- Total subparticles: 18000
- Template: SP-002-CAL1 weak-plane x-normal bonded Li4SiO4 pebble

This is an intermediate statistical bed before attempting 100+ mother pebbles.
